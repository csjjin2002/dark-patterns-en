## About

This is the English version of the dark pattern labeling usability experiment.

## Experiment Design

Dark Pattern Types: FA (Forced Action), OB (Obstruction), SN (Sneaking), II (Interface Interference), SE (Social Engineering)
Label Levels: L0 (No Label), L1 (Passive Label), L2 (Interactive Label)
Latin Square: 5 sequences x 20 participants = 100 total
